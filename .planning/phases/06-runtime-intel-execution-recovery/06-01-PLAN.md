<threat_model>
## Trust Boundaries
- Installed shell helpers to Codex sandbox: the runner must grant Codex access to every product-home path the prompt instructs it to read or write.
- Runtime diagnostics to operator recovery: missing or vague logs make installed failures hard to diagnose and prolong broken scheduled runs.

## Threat Register
- threat_id: T-06-01
  category: Denial of Service
  component: Codex daily intel execution
  disposition: mitigate
  mitigation_plan: Ensure the Codex command includes explicit access to `JOB_QUEST_DATA_DIR` and `JOB_QUEST_REFERENCES_DIR`, and keep the app root set to `JOB_QUEST_APP_ROOT`.
- threat_id: T-06-02
  category: Repudiation
  component: installed-runner diagnostics
  disposition: mitigate
  mitigation_plan: Log the resolved runtime, app root, data dir, references dir, and failing command path before the helper exits.
</threat_model>
