## Trust Boundaries
- Server date logic to dashboard readiness: if endpoint date selection disagrees across routes, the UI shows partial or stale "today" state.
- Local filesystem artifact naming to API behavior: the server must interpret artifact filenames using the same local-day contract the product communicates to users.

## Threat Register
- threat_id: T-06-03
  category: Integrity
  component: daily artifact selection
  disposition: mitigate
  mitigation_plan: Move every "today" calculation behind one local-date helper and one artifact-selection helper that the routes reuse.
- threat_id: T-06-04
  category: Denial of Service
  component: dashboard readiness state
  disposition: mitigate
  mitigation_plan: Make `/api/job-status` use the same local-day filename logic as the quiz/task endpoints so the dashboard cannot show a false partial/pending state at UTC boundaries.
</threat_model>
