## Trust Boundaries
- Runtime contract to implementation phases: ambiguous path or command semantics will cause data-loss risk or runtime drift when installers and scripts are updated.
- Legacy Claude installs to neutral shared home: migration rules must preserve existing user data and scheduler behavior while changing the canonical home.

## Threat Register
- threat_id: T-01-03
  category: Tampering
  component: shared runtime config contract
  disposition: mitigate
  mitigation_plan: Specify exact config keys, allowed runtime values, path semantics, and command-resolution fields instead of prose-only guidance.
- threat_id: T-01-04
  category: Denial of Service
  component: runtime switching and migration
  disposition: mitigate
  mitigation_plan: Define no-breakage legacy handling, automatic runtime-switch persistence, and the exact shared-home layout so scheduler and dashboard paths remain resolvable.
</threat_model>
